    部署說明

    環境要求

    • Node.js >= 18.0.0
    • npm >= 9.0.0
    • SQLite >= 3.0.0

    部署步驟

    # 1. 安裝依賴 npm install # 2. 設置環境變數 cp .env.example .env # 3. 資料庫初始化 npx prisma migrate dev npx prisma db seed # 4. 建構專案 npm run build # 5. 啟動服務 npm start

    API 文檔

    訪問 http://localhost:3000/api-docs 查看完整的 API 文檔。
